# SQL Project

##  Overview
This repository contains SQL scripts that demonstrate the design and implementation of a relational database. It includes table creation, data insertion, and various queries.

##  Features
- Table creation using DDL commands
- Data insertion using DML
- Joins (INNER, LEFT, RIGHT)
- Aggregate functions (SUM, AVG, COUNT)
- Subqueries and nested queries
- Views and constraints

##  Technologies Used
- SQL
- MySQL

##  Project Structure
- `README.md` – Project documentation
- SQL files – Queries and database scripts

##  Purpose
This project is created for practicing SQL and understanding DBMS concepts like normalization, relationships, and query optimization.

##  How to Use
1. Install MySQL or any SQL tool
2. Run the SQL scripts
3. Analyze queries and outputs
